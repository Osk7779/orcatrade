# 01 · Identity

The OrcaTrade visual identity is a balance: serif elegance for trust (Cormorant Garant), monospace for technical credibility (Geist Mono), and a navy-ivory-gold palette that reads as "infrastructure, not startup".

## Logo

The current OrcaTrade logo lives at the repo root: [`/orcatrade_logo.png`](../../orcatrade_logo.png).

Used in:
- Site nav (every page header)
- Email transactional sender artwork
- Social profile avatars (LinkedIn, X)
- PDF report cover page

### Variants needed (gap, to be commissioned)

The kit currently ships with a single PNG variant. The brand kit needs:

- [ ] **SVG primary** — vector source, full-colour
- [ ] **Mono dark** (single-colour navy `#0a1628` on light backgrounds)
- [ ] **Mono light** (single-colour ivory `#f5efe2` on dark backgrounds)
- [ ] **Wordmark only** (no glyph, for tight horizontal spaces)
- [ ] **Glyph only** (no wordmark, for square avatars)
- [ ] **Favicon set** (16 / 32 / 64 / 192 / 512 px)

## Clearspace

Minimum clearspace around the logo = the height of the "O" character × 0.5.

Don't pack the logo into a corner. Don't crop the glyph. Don't put the logo on a busy background without the navy panel beneath it.

## Do

- Place the logo on solid navy (`#0a1628`), ivory (`#f5efe2`), or pure white (`#ffffff`).
- Maintain proportions (no stretching).
- Use the SVG when the platform allows it (sharper rendering, smaller file).

## Don't

- Recolour the logo arbitrarily. Use only the approved variants.
- Add drop shadows or glows.
- Rotate or skew the logo.
- Place the logo on imagery without a backing panel.
- Combine the logo with another company's logo without 1× clearspace between them.

## Co-branded contexts

When OrcaTrade appears alongside a partner brand (forwarder logo, regulatory body reference, customer logo wall), maintain equal visual weight (matched height) and 1× clearspace between marks.
