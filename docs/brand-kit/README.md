# OrcaTrade Brand Kit

> The single source of truth for everything visual, verbal, and structural about the OrcaTrade brand. Owned by the Brand & Creative Director (employee #6 in [marketing-ops.md](../marketing-ops.md)). Read by everyone else.

If you are about to publish anything OrcaTrade-branded — a blog post, a LinkedIn carousel, an ad creative, a slide, an email — you start here.

## Versioning

This kit lives in git. Every change is auditable. The Brand & Creative Director is the only writer; pull requests from other roles get a single review pass.

When the brand evolves (new colour primary, new tagline, new ICP focus), bump the version at the bottom of the relevant file with a one-line changelog entry.

## When something isn't in the kit

If you're publishing something the kit doesn't cover, you have two options:

1. **Use the closest existing template** and flag a brand-kit gap to #6 in the Friday retro.
2. **Ask #6 directly** before improvising — small, fast decisions, not committee work.

Never publish branded work that improvises on identity (logo treatment), colour (off-palette hex), or messaging (positioning sentence). The kit is non-negotiable on those three.
