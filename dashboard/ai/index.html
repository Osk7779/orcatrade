What this shows: every Anthropic API call from a production agent emits a structured ai_call event ({ agent, promptVersion, model, inputTokens, outputTokens, cacheReadTokens, costCents, latencyMs, stopReason }). The dashboard reads the most recent ones via /api/audit?type=ai_call. No raw PII — events carry only the SHA-256 first-16-hex emailHash pseudonym (per ADR 0008) alongside agent + prompt + model + token counts. The per-tenant panel groups by that pseudonym. Pricing source: anthropic.com/pricing, frozen in lib/ai/cost-telemetry.js for deterministic offline cost math.
